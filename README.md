# 市场营销 - marketing

面向市场营销团队的全域营销工具集。覆盖SEO优化、品牌审查、广告合规、活动策划、社媒热点、竞品追踪、营销分析和文案撰写，支持从策略到执行的全链路营销工作。

## 子技能列表

| 子技能 | 功能 | 触发关键词 |
|--------|------|-----------|
| SEO内容优化 | 输入网站URL或文章内容，进行全面SEO诊断 | SEO优化, SEO, 关键词优化, 搜索排名 |
| 品牌调性审查 | 逐段审查内容与品牌调性的一致性 | 品牌审查, 品牌调性, brand review |
| 广告合规检查 | 对照广告法逐句扫描极限词和违规表述 | 广告合规, 极限词, 广告法 |
| 活动策划 | 输入活动目标和预算，输出完整活动方案 | 活动策划, 营销活动, campaign |
| 社媒热点追踪 | 获取当日热点话题，评估借势安全性 | 社媒趋势, 热点追踪, 社交监听 |
| 竞品追踪 | 追踪竞品动态，输出功能对比和SWOT分析 | 竞品跟踪, 竞品监控, 竞品动态 |
| 营销效果分析 | 输入各渠道投放数据，输出营销效果分析报告 | 营销分析, ROI, 获客成本 |
| 营销文案 | 输入产品信息，生成适配平台的营销文案 | 文案撰写, 文案, copywriting |

## 使用方法

通过 Marvis 对话自然触发，说出需求即可自动匹配对应子技能。
